Общее описание

Приложение GPLVote Sign Doc дает возможность пользователю простым образом использовать свою криптографическую цифровую подпись для подписания различных документов. Документы для подписания могут создаваться различными сайтами и использоваться для различных целей: для подтверждения действий пользователя на сайте (например, смены пароля), для подписания различных петиций или для участия в голосованиях.

 

Использование такой подписи делает невозможной подделку подписанных данных пользователя. Что очень важно для действий, влекущих за собой существенные последствия. Например, несанкционированная спена пароля пользователя на сайте может привести к потере доступа к этому сайту. Это не существенно для новостного сайта. Но если это сайт интернет магазина или платежной системы? Тогда речь может идти о вполне реальной денежной потере.

 

Особенности нашего приложения

 

Суть предлагаемого решения заключается в использовании специального мобильного приложения. В нем формируются ключи шифрования, которые в последующем используются для подписания документов. Особенности этого процесса в том, что в нем участвует два ключа - открытый, доступный всем, и закрытый, который храниться в тайне в данном приложении. По открытому ключу вас будут идентифицировать на сайтах и других системах. И с его-же помощью будут проверять вашу подпись на документах. С помощью закрытытого ключа в приложении вы можете подписывать документы.

 

И самое главное заключается в том, что с помощью открытого ключа можно только проверить вашу подпись. А создать ее можно исключительно с помощью закрытого ключа, доступ к которому имеете только вы. Сила этой защиты строиться на математике простых чисел. На современном уровне развития компьютерной техники эти методы подписания документов являются практически полностью надежными. А открытость исходных кодов приложения позволяет гарантировать отсутствие нем других лазеек.
